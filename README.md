# 🚀 Ethereum Builder Pipeline – Bhopal

This repository represents our application and technical intent for participating in the **Ethereum Builder Pipeline – Bhopal**, part of the Road to Devcon ecosystem.

We are not approaching this as a hackathon.  
We are approaching this as an entry point into **serious Ethereum development and contribution**.

---

## 🧠 Context

The Ethereum ecosystem is moving fast — but the gap between *learning* and *shipping real products* is still huge.

Programs like the Road to Devcon University Pipeline aim to bridge that gap by:
- Identifying builders early
- Supporting real-world projects
- Creating pathways to Devcon-level exposure

This repository is our commitment to building within that ecosystem.

---

## 🎯 What We Applied For

### 1. 🏫 University Program
